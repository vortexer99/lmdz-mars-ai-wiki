辐射计算主题

主题目标

辐射计算是 LMDZ.MARS 物理包的核心过程之一，负责计算火星大气中的长波（热红外）和短波（太阳辐射）辐射传输过程。该主题的目标是：

  1. 计算大气层的辐射加热率和冷却率
  2. 计算地表和大气顶的辐射通量
  3. 处理气溶胶（尘埃、水冰、CO2冰）的散射和吸收
  4. 为大气动力学提供辐射强迫

源码依据：callradite_mod.F 注释说明 "Subroutine designed to call the main canonic radiative transfer subroutine 'lwmain' et 'swmain' to compute radiative heating and cooling rate and radiative fluxes to the surface."

入口

辐射计算的主入口是 callradite() 子例程，位于 libf/phymars/callradite_mod.F。

调用位置：在 physiq_mod.F 的 physiq() 主物理例程中被调用。

源码依据：callradite_mod.F 第 7-15 行定义了子例程接口。

计算阶段

阶段 1：初始化（firstcall）

  1. 识别气溶胶种类（尘埃、水冰、CO2冰）
  2. 加载光学属性查找表（SUAER）
  3. 设置维度参数

阶段 2：气溶胶粒径更新

  1. 调用 updatereffrad() 更新各类气溶胶的粒径分布
  2. 尘埃：根据质量和数量混合比计算几何平均半径
  3. 水冰：根据微物理方案或典型值计算
  4. CO2冰：根据微物理方案计算

阶段 3：光学属性计算

  1. 调用 aeroptproperties() 计算3D散射参数
  2. 使用对数正态分布和高斯-勒让德积分
  3. 计算可见光和红外的消光效率、单次散射反照率、不对称因子

阶段 4：光学厚度计算

  1. 调用 aeropacity() 计算各层各类气溶胶的光学厚度
  2. 处理尘埃场景（Viking、MGS、TES等）
  3. 处理沙尘暴和地形尘埃
  4. 计算尘埃缩放系数

阶段 5：长波辐射计算

  1. 计算普朗克函数（lwb）
  2. 计算吸收物质量（lwu）
  3. 计算传输函数（lwxd、lwxn、lwxb）
  4. 计算冷却率和通量（lwflux）
  5. 应用半隐式格式（lwi）

阶段 6：短波辐射计算

  1. 计算火星太阳常数
  2. 循环两个太阳光谱区间
  3. 调用 Fouquart 或 Toon 算法
  4. 计算加热率

阶段 7：输出整理

  1. 将子域结果合并到全局数组
  2. 设置高层大气（nlaylte以上）的加热率为零
  3. 输出诊断信息

与其他主题的耦合

与尘埃循环的耦合

  • 尘埃粒径分布由尘埃循环模块提供
  • 尘埃光学厚度影响辐射加热
  • 辐射加热反过来影响大气动力和尘埃输送

源码依据：callradite_mod.F 使用 dust_param_mod 中的 doubleq、active、submicron 开关。

与水循环的耦合

  • 水冰粒径由水循环模块提供
  • 水冰云影响辐射传输
  • 辐射冷却影响水冰凝结

源码依据：callradite_mod.F 使用 callkeys_mod 中的 water、activice 开关。

与CO2循环的耦合

  • CO2冰粒径由CO2循环模块提供
  • CO2云影响辐射传输
  • 辐射冷却影响CO2凝结

源码依据：callradite_mod.F 使用 callkeys_mod 中的 co2clouds、activeco2ice 开关。

与边界层的耦合

  • 辐射加热率影响边界层结构
  • 边界层湍流影响尘埃混合

与动力核心的耦合

  • 辐射加热率作为动力核心的强迫项
  • 动力核心提供风场用于尘埃输送

输出结果

直接输出

  1. 长波加热率（dtlw）：K/s，用于大气温度演化
  2. 短波加热率（dtsw）：K/s，用于大气温度演化
  3. 地表通量：用于地表能量平衡
  4. 大气顶通量：用于行星能量收支

诊断输出

  1. 气溶胶光学厚度（tau）：用于与观测比较
  2. 气溶胶粒径（rdust、rice、riceco2）：用于微物理诊断
  3. 密度缩放光学厚度（dsodust）：用于与TES观测比较

伪代码总览

SUBROUTINE callradite(...)
  ! 1. 初始化（firstcall）
  IF (firstcall) THEN
    ! 识别气溶胶种类
    ! 加载光学属性查找表
    CALL SUAER
  END IF

  ! 2. 更新气溶胶粒径分布
  CALL updatereffrad(...)

  ! 3. 计算3D散射参数
  CALL aeroptproperties(...)

  ! 4. 计算光学厚度
  CALL aeropacity(...)

  ! 5. 子域循环
  DO jd = 1, ndomain
    ! 分割输入变量

    ! 6. 长波辐射计算
    CALL lwmain(...)

    ! 7. 短波辐射计算
    cste_mars = flux_1AU / (dist_sol * dist_sol)
    CALL swmain(...)

    ! 8. 合并输出变量
  END DO

  ! 9. 设置高层大气加热率为零
  IF (nlayer > nlaylte) THEN
    dtlw(:, nlaylte+1:nlayer) = 0.
    dtsw(:, nlaylte+1:nlayer) = 0.
  END IF
END SUBROUTINE

复现检查点

必须保留的逻辑

  1. 气溶胶种类识别：必须正确匹配 callphys.def 中的 naerkind 和 name_iaer
  2. 光学属性查找表：必须使用正确的查找表文件
  3. 子域分割：必须正确处理 ndomainsz 维度
  4. LTE层限制：必须使用 nlaylte 限制辐射计算范围

关键公式

  1. 火星太阳常数：cste_mars = flux_1AU / (dist_sol * dist_sol)
  2. 光学厚度计算：aerosol = 0.75 * QREFvis / (rho * reffrad) * mass * dp / g
  3. 加热率计算：PHEAT = gcp * ZDFNET / PDP

验证检查点

  1. 光学厚度合理性：待确认：典型值 0.1-5.0（可见光）需补充观测、文档或测试依据
  2. 加热率量级：待确认：长波 ~1 K/solar day、短波 ~1-10 K/solar day 的量级需补充源码、文档或测试依据
  3. 能量守恒：待确认：大气顶入射短波、反射短波、向上长波与地表吸收的闭合检查需数值验证
