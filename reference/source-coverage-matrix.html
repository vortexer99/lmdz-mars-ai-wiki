Source Coverage Matrix

Purpose

This page tracks source-file coverage for the LMDZ.MARS wiki. It is a project-management coverage matrix, not a reader-facing source index. reference/file-index.md remains the reader-facing index for pages that already exist.

Scan Scope

  • Source root: LMDZ.MARS
  • Included files: *.F, *.F90, *.f, *.f90
  • Matrix management scope: the existing 233-row scope used by TASK-0052/TASK-0070, covering libf/aeronomars, libf/dynphy_lonlat, matrix-managed libf/phymars, libf/phymars/dyn1d, and selected util sources. Full repository expansion is intentionally left to a separate task.
  • Total matrix source files: 233
  • Wiki Markdown pages scanned: 262
  • Source-file existence check: passed
  • Direct coverage rule: has_page now requires an exact-stem file page, an explicit helper/routine page that is the primary reproduction page for that source, or an explicit directory page for a small multi-file tool. Theme pages, directory overview mentions, and reference/file-index.md mentions alone count only as mentioned_only.
  • Status definitions:
    • has_page: a wiki page directly documents this file, helper pair, routine, or small tool directory enough to reproduce the source logic.
    • mentioned_only: the file is mentioned by an existing page but does not appear to have its own direct reproduction page.
    • uncovered: no wiki mention was found by filename/stem search.
