热层化学反应速率常数参考

路径

LMDZ.MARS\deftank\chemthermos_reactionrates.def

类型

参考数据——火星热层/电离层化学网络的温度依赖反应速率常数。

作用

源码依据（libf/aeronomars/chemthermos_readini.F:7-40、paramfoto_compact.F:1359-1892）：本文件为 LMDZ.MARS 的热层气相化学网络提供 61 个双体反应的 Arrhenius 速率常数参数。这些参数驱动 chemthermos 模块（热层光化学 + 离子化学）计算中性气体、自由基、激发态原子与离子的源汇项。

文件结构

顶层结构（源码依据：chemthermos_readini.F:23-30）

行 1:       文件头说明行（Fortran list-directed read 读取时被跳过）
行 2-4:     反应 #1 (ch2)
  行 2:     ! ch2: <反应方程式>              ← 跳过
  行 3:     ! <来源/文献>                    ← 跳过
  行 4:     <rc1>  <rc2>  <rc3>              ← 读入 rcoef(1,1:3)
行 5-7:     反应 #2 (ch3)
  ...
行 3n-1..3n+1:  反应 #n
  ...
行 182-184: 反应 #61 (ch87)

共 61 个反应块 × 3 行/块 + 1 文件头 = 184 行（与 wc -l 实测一致）。

速率常数公式（源码依据：文件首行注释 + paramfoto_compact.F:1505-1892）

每个反应 i（Fortran 索引 rcoef(i, 1:3)）在中性温度 T (K) 或电子温度 T_e (K) 下的体积反应速率为：

k(T)  =  rcoef(i,1) * (T/300)^rcoef(i,2) * exp( rcoef(i,3) / T )
  • rcoef(i,1) ≡ A：300 K 参考速率（单位依反应阶数而定：双体 cm³ s⁻¹；三体 cm⁶ s⁻¹；电子反应 cm³ s⁻¹）。
  • rcoef(i,2) ≡ B：温度指数（无量纲）。
  • rcoef(i,3) ≡ C：活化温度（K）；负值表示吸热反应。

复现要求

  • 反应数严格为 61（源码依据：chemthermos_readini.F:23 do i=1,61、param_v4_h.F90:45 rcoef(61,3)）。任何增删都会导致数组越界或读入错误。
  • 每反应严格占 3 行（跳过/跳过/读值）：不能插入额外空行或注释，否则后续所有反应的系数将错位。
  • 速率公式符号约定：活化温度 C 以 exp(C/T) 形式出现（不是 JPL 习惯的 exp(-E_a/T)）。因此吸热反应的 C 在本文件中为负值（如 ch15 C=-1800 K 表示 E_a/k ≈ 1800 K）。重新实现时必须保持此约定，否则会反转吸热/放热方向。
  • A 的单位：双体反应为 cm³ molecule⁻¹ s⁻¹，三体反应为 cm⁶ molecule⁻² s⁻¹。重新实现时必须确保与 chemthermos 模块中其他数密度的单位一致（推断为 CGS）。
  • 温度选择分支：rcoef(1..33) 用 tcte，rcoef(34..61) 用 t_elect。若在重新实现时混淆此分支，离子反应的速率差可达数个数量级。
  • ch56 (rcoef 40) 的 A=0 必须保留：源码依赖此反应被"禁用"来规避 Kim 1989 文献数据的疑虑；若重新实现时用非零值，会与历史结果产生系统性偏差。
  • 复现风险：本文件引用大量 JPL 2011 与 UMIST 数据，但未标注版本年份或检索日期。JPL 数据每 2-4 年更新（最新为 JPL 2019），重新实现时若采用更新后的推荐值，关键反应（如 ch2/ch3/ch4）可能与本文件不一致。
  • 复现风险：部分反应来源为 "UMIST database"，未指明具体版本或反应编号——UMIST 2012 与 UMIST 2021 对同一反应速率的推荐值有时相差数倍。
  • 复现风险：因子 ×2.5（ch2/ch21/ch40）的物理解释未在本文件明确给出，推断为 CO₂ 第三体效率修正。重新实现时需要查阅 JPL 2011 Evaluation 17 的 Table 4-1 或类似权威来源确认。
  • 复现风险：ch6 (ch9, o(1d)+h2o→2oh, C=+60) 与 ch19 (ch19, o(1d)+co2→o+co2, C=+115) 的 C 为正值——表示随温度升高速率增加，与典型吸热反应规律相反；JPL 2011 对这两个反应的推荐形式为 k = A + B·exp(-C/T) 或 A·(T/300)^B（无 exp 项），本文件的 exp(C/T) 形式与原文献不完全一致，可能是拟合近似。
