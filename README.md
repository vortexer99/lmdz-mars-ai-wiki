# LMDZ.MARS Wiki HTML

This repository contains the generated static HTML version of an unofficial LMDZ.MARS code-logic wiki.

The site is intended for reading and navigation only. It does not include the LMDZ.MARS Fortran source tree, the Markdown wiki source, task reports, build scripts, datasets, start files, PDFs, or generated work logs.

Open `index.html` locally, or publish this repository with GitHub Pages from the repository root.

## Scope

- Static HTML pages generated from local wiki notes.
- CSS assets required by the generated HTML.
- Minimal repository metadata and upstream license notices.

## Upstream Project

LMDZ.MARS is the Mars Planetary Climate Model physics package. The upstream source tree used as the documentation target includes `Licence_CeCILL_V2-en.txt` and `Licence_CeCILL_V2-fr.txt`; copies are included under `UPSTREAM-LICENSES/`.

This repository is not an official LMDZ.MARS distribution and is not affiliated with or endorsed by the LMDZ.MARS maintainers.
