MARS-COMMON 文件/功能映射表

页面状态：初版，基于文件名、路径转发文件和 USE/CALL 搜索建立。
源码范围：LMDZ.MARS 与 LMDZ.COMMON-6.3\LMDZ.COMMON。 用途：从 MARS 侧反查 COMMON 的公共动力、接口和支撑文件。

关系类型

关系 含义
direct-path-forward MARS 文件内容只有一行相对路径，直接指向 COMMON 文件；这是最高置信关系。
planet-specific-variant MARS 与 COMMON 中 Venus/Titan 等目录存在同名文件，表示同类流程的行星专用实现，不等价。
common-provider MARS 文件通过 USE 或 CALL 使用 COMMON 模块/例程。
same-name-candidate 仅由同名发现，需后续文件页核验。
no-direct-match 当前未发现直接 COMMON 对应文件。

Mars-only 范围说明

planet-specific-variant 表中的 Venus/Titan 文件只用于帮助读者理解“同名但不同星球实现”的边界，不作为待写任务。Mars 运行实际应优先写 MARS phymars 使用到的 COMMON 共享接口、路径转发文件和 USE/CALL 提供者。

维护规则

  • 文件页完成后，把“对应 COMMON 任务”列中的任务号补成实际 wiki 页面链接。
  • 发现新的路径转发文件时，优先加入“直接路径转发”表。
  • 仅同名但未读源码的关系必须保持 same-name-candidate 或 planet-specific-variant，不能写成等价实现。
  • 非 Mars 路径 不扩批、不领取；只在本映射表中保留边界说明。
