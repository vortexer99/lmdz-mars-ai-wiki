gcm runtime option entry

页面状态：6.3 返工完成。 输入范围：dyn3d/gcm.F90、dyn3dpar/gcm.F、misc/program_options.F90。 Mars 运行参与度：必经。

调用位置

入口 use 行 call 行 说明
libf/dyn3d/gcm.F90 32 146 串行 GCM。
libf/dyn3dpar/gcm.F 29 160 并行 GCM。

两处调用都发生在 conf_gcm、tracer 和 start/restart 初始化之前。

顺序关系

入口启动
  -> parse_args()
  -> conf_gcm / getin / run.def
  -> tracer/start/restart/history 初始化
  -> leapfrog 或 leapfrog_p

未知 option 会通过 program_options 早期 abort。排查配置未读取时，应先确认命令行参数没有误传。
