dynphy_lonlat 总览

路径

LMDZ.COMMON-6.3\LMDZ.COMMON\libf\dynphy_lonlat

职责

dynphy_lonlat 提供 lon-lat 动力和物理之间的接口例程、动力/物理格点变换、物理几何初始化和 A/B 网格转换。当前目录根部有 9 个文件，另有 phyvenus/ 和 phytitan.old/ 行星专用子目录。

Mars 运行参与度：根部共享接口为 条件经过 或 接口兼容；phyvenus/、phytitan.old/ 为 非 Mars 路径。MARS dyn1d 直接转发 mod_interface_dyn_phys.F90，MARS iniphysiq_mod.F90 使用并调用 COMMON inigeomphy_mod。

目录内容

mod_interface_dyn_phys.F90 接口兼容 动力-物理接口状态模块，MARS dyn1d 直接转发。
grid_atob_m.F90 条件经过 A/B 网格转换、粗细格重采样、海冰/粗糙度插值辅助。

典型流程

  1. 动力入口初始化动力几何和常量后，物理初始化路径调用 iniphysiq。
  2. MARS 物理初始化可通过 COMMON inigeomphy_mod 生成物理侧经纬度、面积和映射信息。
  3. 时间推进中，动力状态通过 calfis 或行星侧同类接口整理为物理可消费的 lon-lat 物理点场。
  4. 物理 tendency 再经格点转换或接口数组返回动力侧。

复现要点

  • 根部共享接口和 phyvenus/phytitan.old 不能混写；同名 newstart/readstart/start2archive 是其他行星变体，不等价于 Mars 实现。
  • gr_dyn_fi 与 gr_fi_dyn 的维度方向相反，详见 动力/物理格点互转。
  • 并行 physics 差异要同时查 calfis_p、mod_interface_dyn_phys 映射和 phy_common 物理侧分布。

待确认

  • grid_atob_m.F90 在 Mars 当前输入数据预处理中的具体调用点需后续输入数据任务核验。
  • calfis.F 与 MARS phymars 物理调用链的直接/间接关系需后续主题页补完整证据。
