phy_common 总览

路径

LMDZ.COMMON-6.3\LMDZ.COMMON\libf\phy_common

职责

phy_common 提供物理侧公共基础设施：物理列并行分布、MPI/OpenMP transfer、物理几何和垂直层状态、XIOS/write_field 封装、打印控制和物理侧 abort。它位于动力-物理接口之后，为行星物理包提供统一的网格、并行和输出支撑。

Mars 运行参与度

条件经过：Mars 物理初始化、物理输出或 COMMON 物理侧并行抽象被使用时经过；具体文件是否进入 Mars 标准路径取决于 phymars 调用和编译宏。

复现要点

  • klon_glo 是全局物理列数，去掉重复极点经度；klon_mpi 是当前 MPI rank 的列数；klon_omp 是当前 OpenMP 线程的列数。
  • mod_grid_phy_lmdz 只保存全局网格规模和 1D/2D 映射；经纬度、面积和边界数组由 regular_lonlat_mod 与 geometry_mod 保存。
  • geometry_mod 和 vertical_layers_mod 的关键数组是 THREADPRIVATE，必须在 OpenMP parallel 区内初始化。
  • MPI/OMP transfer 组页用于快速定位；MPI transfer、OMP transfer 和 para wrapper hard 父文件页均已完成。
  • 输出和打印控制由本目录的专页展开，目录页只保留职责分组。

待确认
