抽象层

mod_fft.F90 是编译期选择器：

  • FFT_MATHKEISAN 时使用 mod_fft_mathkeisan。
  • FFT_FFTW 时使用 mod_fft_fftw。
  • 其他 MKL 构建可使用 mod_fft_mkl 和 MKL_DFTI 接口。
  • 无 backend 时落到 mod_fft_wrapper，其 Init_fft/fft_forward/fft_backward 直接 STOP，提示必须用 CPP key 指定 FFT。

滤波 FFT 流程

mod_filtre_fft 和 mod_filtre_fft_loc 都提供：

  1. Init_filtre_fft：根据 U/V 滤波系数和南北滤波纬带构造 spectral multiplier，并调用 Init_fft。
  2. Filtre_u_fft、Filtre_v_fft、Filtre_inv_fft：打包待滤波数组，调用 FFT_forward，按 mode multiplier 修改谱空间结果，再 FFT_backward。
  3. local 版本显式接收纬向本地范围，适配并行 decomposition。
