滤波主流程和并行滤波调用组页

初始化

filtreg_mod 保存 matriceun/matriceus/matricevn/matricevs/matrinvn/matrinvs 等滤波矩阵。初始化时：

  1. 调用 inifgn 计算离散 Laplacian 的特征函数和特征值。
  2. 按 jfiltnu/jfiltsu/jfiltnv/jfiltsv 检查南北半球滤波纬带。
  3. 使用 MXM 或 SGEMM 构造 U/V 和 inverse 滤波矩阵。
  4. 若 use_filtre_fft，调用 Init_filtre_fft 和 local 版本初始化 FFT 系数。

串行 filtreg

filtreg(champ,nlat,nbniv,ifiltre,iaire,...) 根据 ifiltre 和 iaire 选择 U/V 或 inverse 矩阵，对每个需滤波纬带做矩阵乘法。ifiltre 不是 2 或 -2 时打印错误；源码注释提示某些模式应使用旧滤波路径。

并行 filtreg_p

并行版本额外接收 ibeg/iend/nlat 子域信息：

  • 若未启用 FFT，按本地纬向范围对 champ_fft 临时数组执行 DGEMM 矩阵滤波。
  • 若启用 FFT，先复制本地场到 champ_fft，再按 iaire 调 Filtre_inv_fft、Filtre_u_fft 或 Filtre_v_fft。
  • 结束时将对称/反对称组合写回 champ，并通过 timer_filtre 统计耗时。

复现要点

  • coefils.h 中的 jfiltn*、sddu/sddv 和矩阵数组是滤波纬带与权重的核心状态。
  • 串行和并行矩阵路径都依赖 BLAS SGEMM/DGEMM；并行路径还依赖 OMP_CHUNK 分块。
  • FFT 路径在 filtreg_p 中只在 use_filtre_fft 为真时启用。

待确认

  • Mars 默认配置中 use_filtre_fft 的取值和 conf_gcm 读取路径需由动力配置 hard 页最终确认。
