初始化链

filtreg_mod 调用 inifgn(eignvl)。inifgn：

  1. include dimensions.h、paramet.h、comgeom.h、coefils.h。
  2. 构造 U/V 网格相关矩阵。
  3. 调用 jacobi 求特征向量和特征值。
  4. 调用 acc 调整特征向量。
  5. 调用 eigen_sort 使特征值/特征向量顺序稳定。

coefils.h 通过 common block 保存 jfiltnu/jfiltsu/jfiltnv/jfiltsv、sddu/sddv 及对应 inverse 权重，是滤波矩阵构造和主滤波例程共享的状态。

更细的 filtreg.F 施加流程、ifiltre/iaire/griscal 契约、inifgn -> jacobi/acc/eigen_sort 特征基构造见 filtreg-operators。

计时

timer_filtre 使用 cpu_time 保存 last_time 和累计耗时。并行 filtreg_p 在进入滤波前 start_timer，结束时 stop_timer。
