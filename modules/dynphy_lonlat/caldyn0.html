文件定位

lon-lat GCM 动力倾向计算例程；由 newstart.F 调用，按顺序调用 13 个 EXTERNAL 子过程完成协变/反变风转换、气压计算、空气质量、质量通量、质量辐合、垂直速度、位涡、动能、Bernoulli 函数等，最终调 sortvarc0 输出诊断。

共享状态与副作用

  • 通过头文件 comgeom.h 读取 airesurg（格点面积）和 constang（科氏参数）。
  • 通过 comvert_mod::ap, bp 读取垂直坐标参数。
  • EXTERNAL 子过程（advect, traceur 等）通过 EXTERNAL 声明但部分未在本例程体中直接调用。

核心逻辑

  1. 风分量转换（行 60）：covcont 把协变风 ucov, vcov 转换为反变风 ucont, vcont。

  2. 气压与质量计算（行 61-66）：pression 由 ap, bp, ps 算气压 p；psextbar 提取地表气压水平平均；massdair 计算空气质量；massbar/massbarxy 计算质量加权水平平均；flumass 计算质量通量。

  3. 质量辐合与底层气压倾向（行 67-71）：convmass 计算质量辐合 convm，底层 dp = convm(1)/airesurg。

  4. 垂直速度、位涡、动能、Bernoulli（行 73-76）：依次计算 w, vorpot, ecin, bern = phi + ecin。

  5. 绝对涡度（行 78-82）：ang = ucov + constang（相对涡度 + 科氏参数）。

  6. 诊断输出（行 84-85）：sortvarc0 收集所有诊断变量输出。

伪代码

covcont(ucov, vcov) -> ucont, vcont
pression(ap, bp, ps) -> p
psextbar(ps) -> psexbarxy
massdair(p) -> masse
massbar(masse) -> massebx, masseby
massbarxy(masse) -> massebxy
flumass(massebx, masseby, vcont, ucont) -> pbaru, pbarv
convmass(pbaru, pbarv) -> convm

for each grid point ij:
    dp(ij) = convm(ij, 1) / airesurg(ij)

vitvert(convm) -> w
tourpot(vcov, ucov, massebxy) -> vorpot
enercin(vcov, ucov, vcont, ucont) -> ecin
bernoui(phi, ecin) -> bern

for each (ij, l):
    ang(ij, l) = ucov(ij, l) + constang(ij)

sortvarc0(itau, ucov, teta, ps, masse, pk, phis, vorpot, phi, bern, dp, time, vcov)

参与的主题流程

主题 参与方式
动力核心 计算 lon-lat GCM 动力倾向；由 newstart 在时间循环中调用

写法特点

  • 固定格式 Fortran（.F），续行符 $。
  • 独立子程序（非 module），通过 EXTERNAL 声明引用 13 个子过程。
  • 使用头文件（dimensions.h, paramet.h, comgeom.h）定义网格和物理常量。
  • 部分 EXTERNAL 声明（advect, traceur）在例程体中未被直接调用，可能为遗留声明。
  • 数组维度由头文件宏定义（ip1jmp1, ip1jm, llm 等）。

复现要点

  • 需要 comvert_mod::ap, bp 已初始化。
  • 需要头文件提供正确的网格维度参数。
  • newstart.F 行 2005 为唯一调用方。
  • 所有 EXTERNAL 子过程需在同一链接中可用。
  • sortvarc0 负责最终诊断输出，需确认其具体行为。

待确认

  • EXTERNAL 声明中的 advect 和 traceur 是否曾在本例程中被调用或仅为遗留声明。
  • sortvarc0 的具体输出行为（诊断文件/XIOS）。
  • massbar 和 massbarxy 的精确差异。
  • 与 GCM 动力核心（dynzon 或 dissip）的关系。
