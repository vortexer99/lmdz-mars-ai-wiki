文件定位

垂直坐标线性插值例程。将 GCM 变量从旧垂直离散化方案插值到新的垂直离散化方案，用于混合 σ-p 坐标系下的垂直重映射。典型应用场景包括读取初始场（restart）时，初始文件的垂直层与当前模拟的垂直层不一致，需要插值对齐。

作者：Francois Forget（1995-01），混合坐标修改（2003-03）。

符号 类型 行号 作用
interp_vert subroutine 2 将变量从旧垂直网格线性插值到新垂直网格

依赖的模块

核心逻辑

  1. 对每个水平点 n = 1..Nhoriz：
  2. 计算新网格的 sigma 坐标：sigmn(ln) = aps(ln)/ps(n) + bps(ln)，ln = 1..lmn
  3. 计算旧网格的 sigma 坐标：sigmo(lo) = apso(lo)/ps(n) + bpso(lo)，lo = 1..lmo
  4. 对每个新层 ln = 1..lmn，按 sigma 值在旧网格中定位并插值：
    • 若 sigmn(ln) >= sigmo(1)（新层在旧网格顶部之上）：取旧顶层值 varo(n,1)
    • 若 sigmn(ln) <= sigmo(lmo)（新层在旧网格底部之下）：取旧底层值 varo(n,lmo)
    • 否则：找到 lo 使得 sigmo(lo+1) < sigmn(ln) <= sigmo(lo)，线性插值 varn = varo(lo) + coef * (varo(lo+1) - varo(lo))，其中 coef = (sigmn(ln) - sigmo(lo)) / (sigmo(lo+1) - sigmo(lo))

伪代码

FOR each horizontal point n:
    Compute sigma levels for new grid: sigmn(ln) = aps(ln)/ps(n) + bps(ln)
    Compute sigma levels for old grid: sigmo(lo) = apso(lo)/ps(n) + bpso(lo)
    FOR each new level ln:
        IF new sigma >= top of old grid:
            varn(n,ln) = varo(n, 1)         -- 顶层外推
        ELSE IF new sigma <= bottom of old grid:
            varn(n,ln) = varo(n, lmo)       -- 底层外推
        ELSE:
            Find bracketing old levels lo, lo+1
            coef = linear interpolation weight
            varn(n,ln) = varo(n,lo) + coef * (varo(n,lo+1) - varo(n,lo))

写法特点

  • 固定格式 Fortran 77（.F 后缀但使用 f77 风格缩进与列格式）
  • 无模块依赖：完全自包含，不 USE 任何模块，不使用 INCLUDE
  • 坐标约定：sigma 坐标从上到下递增（sigmo(1) 为顶，sigmo(lmo) 为底），与混合坐标 sigma = a/p_s + b 的定义一致
  • 边界处理：超出旧网格范围时取最近层值（零阶外推），不做外推
  • 混合坐标：同时使用 ap（气压部分）和 bp（sigma 部分），通过地表气压 ps 计算实际 sigma 值

复现要点

  • 输入 apso/bpso 和 aps/bps 分别为旧/新网格的混合坐标系数
  • 需要地表气压 ps 将混合坐标转换为 sigma 坐标
  • 插值为逐点独立（水平方向无耦合），可并行化
