文件定位

独立物理模式（nogcm）的时间积分主循环。在没有完整 GCM 动力核心的情况下，实现 Matsuno forward/backward + leapfrog 时间步进方案，并用牛顿松弛 + 水平混合替代动力过程，模拟气压、温度和 CO₂ 示踪剂的全局重分布。该例程是 nogcm.F90 的核心驱动。

作者：P. Le Van / L. Fairhead / F. Hourdin。SVN $Id: leapfrog.F 1446 2010-10-22$。

  • 文件 I/O：写入 restart.nc（dynredem1）、历史文件（writehist）、动力平均文件（writedynav）

核心逻辑

  1. 确定终止步 itaufin：根据 nday（正=天数，负=步数）、less1day（分数天）、ndynstep（固定步数）计算
  2. 初始化倾向数组为零：dudis/dvdis/dtetadis/dutop/dufi/dvfi/dtetafi/dqfi/dpfi
  3. 初始气压与 Exner：调用 pression + exner_hyb/exner_milieu，并验证 PK 单调递减
  4. 计算参考气压因子 kpd = exp(-phis/(R*200)) 和全局平均 p00d = globaverage2d(kpd)
  5. 定位 CO₂ 示踪剂：遍历 tname 找到 "co2" 的索引 igcm_co2
  6. Matsuno Forward 步（label 1）：
    • 保存前一步变量（scopy × 5）
    • 设 forward=.TRUE., leapf=.FALSE., dt=dtvr
    • 动力倾向置零（nogcm 无动力核心）
    • 判断物理调用 apphys（按 iphysiq 频率）
  7. 物理调用分支（apphys 为真时）：
    • 重新计算气压、Exner、位势
    • 调用 calfis 计算物理倾向
    • 调用 addfi 施加物理倾向
    • nogcm 特有：牛顿松弛气压 → ps += (p0*kpd - ps) * (1 - exp(-dtphys/tau_ps))，tau_ps = 1s
    • nogcm 特有：位温修正 teta *= (ps_new/ps_old)^kappa
    • nogcm 特有：CO₂ 垂直混合（质量加权柱平均替代垂直分布）
    • nogcm 特有：CO₂ 水平混合（牛顿松弛向全局平均，tau_co2 = 10^5 s）
    • nogcm 特有：温度水平混合（牛顿松弛，tau_teta = 10^7 s）
  8. Matsuno Backward / Leapfrog 步（label 2）：
    • 根据 purmats 选择纯 Matsuno 或 Matsuno+Leapfrog 方案
    • Leapfrog 步 dt = 2*dtvr
  9. 输出与 restart：
    • 按 iperiod 频率写动力平均和诊断
    • 按 iecri 频率写历史文件
    • 终止时写 restart.nc（dynredem1）
  10. 时间步进控制：通过 GO TO 1/2 在 forward/backward/leapfrog 之间跳转

伪代码

Compute itaufin from nday / ndynstep / less1day
Zero all tendency arrays
Compute initial pressure, Exner function; verify PK monotonicity
Compute kpd = exp(-phis / (R*200)); p00d = global_average(kpd)
Locate CO2 tracer index (igcm_co2)

LOOP itau = 0 .. itaufin:
    -- Matsuno Forward step --
    Save current state (ucov, vcov, teta, masse, ps) → (...m1)
    Set forward=.T., leapf=.F., dt=dtvr
    Zero dynamic tendencies (nogcm has no dynamics)
    IF physics due (apphys):
        Recompute pressure, Exner, geopotential
        CALL calfis → physical tendencies (dufi, dvfi, dtetafi, dqfi)
        CALL addfi → apply tendencies
        Newtonian relax ps toward p0*kpd (tau=1s)
        Correct teta for ps change
        Vertical-mix CO2 (column mass-weighted mean)
        Horizontal-mix CO2 toward global mean (tau=1e5s)
        Horizontal-mix teta toward global mean (tau=1e7s)

    -- Matsuno Backward / Leapfrog step --
    IF purmats: pure Matsuno (forward→backward→forward...)
    ELSE: Matsuno forward+backward, then leapfrog (dt=2*dtvr)

    IF output due: write history / averages / diagnostics
    IF itau == itaufin: write restart.nc; stop

参与的主题流程

主题 参与方式
独立物理模式（nogcm） 提供完整的时间积分循环，替代 GCM 动力核心
CO₂ 循环（nogcm 近似） 用牛顿松弛和水平混合近似动力输运
初始场与 restart 终止时写 restart.nc

写法特点

  • 固定格式 Fortran 77/90 混合：.F 后缀，f77 风格列格式但包含 f90 USE 和 ! 注释
  • GO TO 控制流：用 label 1（forward 步起点）和 label 2（backward/leapfrog 步起点）实现时间步进跳转
  • SAVE + DATA 初始化：first/dissip_conservative/igcm_co2/airetot（globaverage2d 内）使用 SAVE 保持状态
  • 条件编译：#ifdef CPP_IOIPSL 控制 IOIPSL I/O 功能；未定义时相关输出被跳过
  • nogcm 特有物理：用 3 个牛顿松弛时间尺度（tau_ps=1s/tau_co2=10^5s/tau_teta=10^7s）+ CO₂ 垂直混合替代完整动力核心
  • globaverage2d 内部函数：面积加权全局平均，首次调用计算 airetot 并 SAVE
  • STOP 语句（line 1015）：正常执行到末尾停止（abort_gcm 在非零错误码时才真正中止）

复现要点

  • 需要 nogcm.F90 提供初始化的动力变量场
  • 需要配置 control_mod 中的运行参数（nday/iphysiq/iconser/purmats 等）
  • 需要 calfis → physiq 物理参数化链可用
  • CO₂ 示踪剂名为 "co2"（字符串匹配 tname）
  • 牛顿松弛常数硬编码在例程内：tau_ps=1, tau_co2=1e5, tau_teta=1e7
