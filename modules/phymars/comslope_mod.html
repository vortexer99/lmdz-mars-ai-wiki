文件定位

次网格坡面（subgrid slope）参数化共享状态模块。定义坡面统计分布、坡度边界、天空可视比例等共享变量，并提供初始化、释放和网格平均计算子程序。用于 Mars PCM 的次网格坡面微气候参数化方案。

共享状态与副作用

  • Module 变量（全部 save + !$OMP THREADPRIVATE）：
    • nslope — 坡面 bin 数量
    • iflat — 平坦坡面索引
    • def_slope(nslope+1) — 坡面边界数组（度）
    • def_slope_mean(nslope) — 每个 bin 的平均坡度（度）
    • sky_slope(nslope) — 每个坡面可见天空比例
    • subslope_dist(ngrid,nslope) — 坡面在网格中的分布比例
    • major_slope(ngrid) — 占据网格大部分的次坡面索引
  • OpenMP: 所有共享变量使用 THREADPRIVATE，每个线程独立副本
  • 内存管理: ini_comslope_h 分配，end_comslope_h 释放

核心逻辑

  1. 初始化（ini_comslope_h，行 28-41）：

    • 接收 ngrid 和 nslope_in
    • 分配 5 个数组：def_slope、def_slope_mean、sky_slope、subslope_dist、major_slope

  2. 释放（end_comslope_h，行 45-55）：

    • 检查每个数组是否已分配，若已分配则释放

  3. 网格平均计算（compute_meshgridavg，行 59-105）：

    • 快速路径（行 84-88）：若 nslope==1，直接赋值（无次网格）
    • 一般路径（行 89-103）：
      • 反照率：albedo_meshavg += albedo_slope * subslope_dist
      • tracer：qsurf_meshavg += qsurf_slope * subslope_dist / cos(slope_angle)（坡面→水平投影转换）
      • 发射率：emis_meshavg += emis_slope * subslope_dist
      • 温度：tsurf_meshavg += emis_slope * tsurf_slope^4 * subslope_dist（Stefan-Boltzmann 加权）
      • 最终温度：tsurf_meshavg = (tsurf_meshavg / emis_meshavg)^0.25

伪代码

! 初始化
SUBROUTINE ini_comslope_h(ngrid, nslope_in)
  分配 def_slope(nslope_in+1)
  分配 def_slope_mean(nslope_in)
  分配 sky_slope(nslope_in)
  分配 subslope_dist(ngrid, nslope_in)
  分配 major_slope(ngrid)

! 释放
SUBROUTINE end_comslope_h
  释放所有已分配数组

! 网格平均
SUBROUTINE compute_meshgridavg(ngrid, nq, albedo_slope, emis_slope, tsurf_slope, qsurf_slope, ...)
  IF nslope == 1 THEN
    直接赋值（无次网格）
  ELSE
    DO ig = 1, ngrid
      DO islope = 1, nslope
        albedo_meshavg += albedo_slope * subslope_dist
        qsurf_meshavg += qsurf_slope * subslope_dist / cos(slope_angle)  ! 坡面→水平投影
        emis_meshavg += emis_slope * subslope_dist
        tsurf_meshavg += emis_slope * tsurf_slope^4 * subslope_dist  ! Stefan-Boltzmann 加权
      END DO
      tsurf_meshavg = (tsurf_meshavg / emis_meshavg)^0.25  ! 反推平均温度
    END DO
  END IF

参与的主题流程

主题 参与方式
水循环 坡面微气候影响地表水冰交换（vdifc 调用）
CO2 循环 坡面温度影响 CO2 凝结/升华（co2condens 调用）
辐射计算 坡面反照率和发射率影响辐射收支

写法特点

  • OpenMP 并行: 所有共享变量使用 !$OMP THREADPRIVATE，每个线程独立副本
  • SAVE 属性: 所有 module 变量使用 save，保持调用间状态
  • 条件分配: ini_comslope_h 负责分配，end_comslope_h 负责释放
  • 快速路径: nslope==1 时跳过加权平均，直接赋值
  • 角度投影: tracer 使用 cos(slope_angle) 转换坡面→水平投影
  • Stefan-Boltzmann 加权: 温度平均使用 T^4 加权而非直接平均

复现要点

  • def_slope 长度为 nslope+1（边界），def_slope_mean 长度为 nslope（中心值）
  • qsurf_slope 单位为 kg/m² sloped，输出 qsurf_meshavg 单位为 kg/m² flat（需除以 cos(slope)）
  • 温度平均使用 Stefan-Boltzmann 加权（T^4），不是算术平均
  • 所有变量为 OpenMP THREADPRIVATE，多线程时每个线程独立
