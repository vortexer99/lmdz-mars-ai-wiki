libf/phymars

文件组定位

这两个文件提供 CO2 凝结的热力学基础计算：

  • co2sat.F：给定温度计算 CO2 饱和蒸汽压
  • tcondco2.F90：给定压力和 CO2 质量混合比反推凝结温度

二者基于相同的 James et al. 1992 饱和蒸汽压公式，形式上互为逆运算，是 CO2 循环与 CO2 云微物理的核心热力学入口。

核心逻辑

co2sat

按 James et al. 1992 公式计算饱和蒸汽压：

psat = 1.382 × 10¹² × exp(-3182.48 / t)
  • 对数组 t(1:naersize) 逐元素应用上述公式
  • 参数为固定常数：前置因子 1.382×10¹² Pa，活化能 3182.48 K

tcondco2

从饱和蒸汽压公式反解温度，给定气压和 CO2 混合比：

  1. 计算 CO2 分压：pco2 = q(ig,l) * (mmean(ig,l)/44.01) * p(ig,l)

    • mmean 为当地平均分子量（通过 conc_mod 读取）
    • 44.01 为 CO2 分子量
    • q 为 CO2 tracer 质量混合比 (kg/kg)，调用点传入 pq(:,:,igcm_co2) 或 pq+pdq*ptimestep
    • 公式含义：将质量混合比转换为体积混合比（q * M_mean/M_CO2），再乘以总压得分压

  2. 反解温度：tcond = B / (log(pco2) - A)

    • A = log(1.382×10¹²)
    • B = -3182.48
    • 推导：由 psat = exp(A) × exp(B/t) 得 log(psat) = A + B/t，故 t = B / (log(psat) - A)

  3. 对所有网格点和垂直层循环应用

伪代码

co2sat

对 i = 1 到 naersize:
    psat(i) = 1.382e12 * exp(-3182.48 / t(i))

tcondco2

A = log(1.382e12)
B = -3182.48

对 l = 1 到 nlay:
    对 ig = 1 到 ngrid:
        pco2 = q(ig,l) * (mmean(ig,l) / 44.01) * p(ig,l)
        tcond(ig,l) = B / (log(pco2) - A)

写法特点

  • co2sat.F 为固定格式 Fortran（F77 风格），tcondco2.F90 为自由格式（F90 风格）
  • 参数为硬编码常数，未从配置文件读取
  • 精度差异：tcond 输出为 double precision，co2sat 输出为 real
  • 调用点核验：co2cloud_mod.F90:515 传入 zq_co2vap = pq(ig,l,igcm_co2) + pdq*ptimestep；improvedco2clouds_mod.F90:316 传入 zq(:,:,igcm_co2) + zq(:,:,igcm_co2_ice)——均为 tracer 质量混合比 kg/kg

复现要点

  • 确保 conc_mod 已正确初始化并提供 mmean 数组
  • tcond 输入 q 为 CO2 tracer 质量混合比 (kg/kg)，经调用点 co2cloud_mod.F90:515 和 improvedco2clouds_mod.F90:316 核验
  • 反解过程中 pco2 不能为零或负值，否则 log(pco2) 将出错（源码未含保护）
  • 参数 A、B 的常数源自 James et al. 1992，修改需同步两文件

待确认

  • pco2 单位：推断为 Pa（与 p 同单位），与 co2sat 输出单位一致
  • tcond 是否需要在 pco2 接近零时设置保护（源码未含保护）

复现风险

  • 当 pco2 ≤ 0 时 tcond 中的 log(pco2) 会崩溃，需确认调用点是否有前置保护
  • 参数 A、B 的精度敏感性：直接使用双精度计算，但 co2sat 仅返回单精度 real
