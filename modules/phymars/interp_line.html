文件定位

interp_line.F 定义 interp_line_mod 模块和 interp_line 子程序，是一个通用的线性插值工具例程。它被 soil_settings.F 和 lect_start_archive.F 调用，用于在有序的源数据点之间进行线性插值，或在目标点超出源数据范围时使用最近的边界值。

本例程的核心功能是：给定源数据 (x1, y1) 和目标横坐标 x2，计算对应的插值结果 y2。当 x2 超出 x1 的范围时，不做外推，而是使用最近的边界值。

核心逻辑

  1. 对每个目标点 x2(i)：
    • 边界检查：若 x2(i) 超出 x1 的范围（即 (x2(i) ≤ x1(1) 且 x2(i) ≤ x1(len1)) 或 (x2(i) ≥ x1(1) 且 x2(i) ≥ x1(len1))）：
      • 使用最近的边界值：若 |x2(i) - x1(1)| < |x2(i) - x1(len1)|，则 y2(i) = y1(1)，否则 y2(i) = y1(len1)
    • 内部插值：否则，找到包含 x2(i) 的区间 [x1(j), x1(j+1)]，进行线性插值： 
      y2(i) = ((x2(i) - x1(j)) / (x1(j+1) - x1(j))) * y1(j+1)
      + ((x2(i) - x1(j+1)) / (x1(j) - x1(j+1))) * y1(j)

伪代码

interp_line(x1, y1, len1, x2, y2, len2):
    for i in 1..len2:
        if x2(i) is outside [x1(1), x1(len1)]:
            if |x2(i) - x1(1)| < |x2(i) - x1(len1)|:
                y2(i) = y1(1)
            else:
                y2(i) = y1(len1)
        else:
            for j in 1..len1-1:
                if x2(i) is in [x1(j), x1(j+1)]:
                    y2(i) = linear_interpolation(x1(j), y1(j), x1(j+1), y1(j+1), x2(i))

写法特点

  • 固定格式 Fortran（.F 扩展名），使用传统 Fortran 77 风格的注释和续行符 &。
  • 纯工具例程，无外部依赖。
  • 使用简单的嵌套循环实现，时间复杂度为 O(len2 × len1)。
  • 边界处理使用最近邻方法，不做外推。
  • 假设 x1 是有序的，但未显式检查。

复现要点

  • 必须确保 x1 是有序的（单调递增或递减）。
  • 当 x2(i) 超出范围时，使用最近的边界值，不是外推。
  • 插值公式使用标准的线性插值，但写法略有冗余。

待确认

  • 是否需要检查 x1 的单调性。
  • 是否可以优化为二分查找以提高性能。
