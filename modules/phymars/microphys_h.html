文件定位

microphys_h.F90 定义 microphys_h 模块，是水冰和 CO2 云微物理共享的常数与线程私有状态头文件。它本身不含例程、不读配置、不执行计算；下游云方案、成核、增长阻力、质量传输和地表 HDO 通量例程通过 use microphys_h, only: ... 读取这些常数，或读写其中少量 SAVE 状态。

源码可分为两段：

  1. H2O/HDO/CO2/N2 的通用分子常数、水冰成核常数，以及水云微物理 bin 数和共享半径数组。
  2. CO2 云微物理常数、CO2 成核 bin 数、CO2 半径数组和接触参数。

共享状态与副作用

  • mteta、vo1、rad_cld、vo1co2、rad_cldco2 都是 SAVE 变量，并声明为 OpenMP THREADPRIVATE。
  • microphys_h 自身没有副作用；副作用来自下游初始化例程写这些共享变量。
  • conf_phys.F 行 863-872 默认 mteta=0.95，若 temp_dependent_m=.false. 则从 callphys.def 读 mteta。
  • improvedclouds_mod.F90 行 143-160 构造水云半径网格，行 182 写 vo1。
  • co2cloud_mod.F90 行 323-336 构造 CO2 半径网格，行 403-404 把 CO2 bin 边界取对数后保存到本地 rb_cldco2。
  • 复现风险：直接调用 nuclea 或 nucleaco2 前，必须确保对应云方案 firstcall 已经给 rad_cld 或 rad_cldco2 赋值，否则成核率会依赖未初始化的线程私有数组。

核心逻辑

  1. 声明水云微物理 bin 数、参考温度、分子常数和水冰成核参数。
  2. 暴露水冰接触参数、单分子体积和半径数组，供 conf_phys / improvedclouds 初始化，供 nuclea 等例程读取。
  3. 声明 CO2 云微物理 bin 数、CO2 成核/扩散常数和固定接触参数 mtetaco2。
  4. 暴露 CO2 单分子体积和半径数组，供 CO2 云路径初始化并供 nucleaco2、improvedco2clouds、co2cloud 使用。

伪代码

module microphys_h:
    define water/cloud constants:
        nbin_cld = 5
        To, nav, rgp, kbz
        molecular masses and effective radii
        water nucleation constants
        m0 = mh2o / nav

    declare threadprivate mutable water state:
        mteta
        vo1
        rad_cld(nbin_cld)

    define CO2 microphysics constants:
        nbinco2_cld = 100
        sigco2, desorpco2, nusco2, surfdifco2
        m0co2 = mco2 / nav
        mtetaco2 = 0.95

    declare threadprivate mutable CO2 state:
        vo1co2
        rad_cldco2(nbinco2_cld)
        threshJA = 1

参与的主题流程

主题 参与方式
水循环 为 improvedclouds、nuclea、growthrate、hdo_surfex 提供分子常数、水冰半径 bin 和接触参数。
CO2 循环 为 co2cloud、improvedco2clouds、nucleaco2、massflowrateco2 提供 CO2 成核/增长常数和半径 bin。
尘埃循环 dust_coagulation_mod 读取 kbz 和 m0co2，CO2 / 水云 CCN 也通过尘埃 tracer 与微物理常数耦合。

写法特点

  • 这是头文件式 Fortran module：没有 contains、没有例程、没有本地初始化过程。
  • 参数常数和可写共享状态混在同一个模块内；读者需要区分 PARAMETER 常数和 SAVE THREADPRIVATE 状态。
  • mh2o、mhdo、mco2、mn2 注释为 kg.mol-1，但 m0=mh2o/nav 和 m0co2=mco2/nav 才是单分子质量。
  • mtetaco2 注释保留了 Bachnar 2016 值 0.78 和旧值 0.95，源码当前实际采用 0.95。

复现要点

  • 水云成核路径需要先经过 conf_phys 初始化 mteta，再由 improvedclouds firstcall 初始化 rad_cld 和 vo1。
  • CO2 成核路径需要先经过 co2cloud firstcall 初始化 rad_cldco2；improvedco2clouds 在尘埃、流星尘和水冰 CCN 分支中读取该数组。
  • THREADPRIVATE 表示并行线程各有一份变量；复现 OpenMP 运行时要保证每个线程都经历了相同初始化路径。
  • microphys_h 不读取 callphys.def；配置只通过 conf_phys 间接写入 mteta。
