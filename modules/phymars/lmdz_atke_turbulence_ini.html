文件定位

lmdz_atke_turbulence_ini.F90 是 ATKE 湍流方案的初始化与参数共享模块。它定义 lmdz_atke_turbulence_ini 模块和唯一入口 atke_ini，从 physiq_mod.F 在 callatke 打开时调用，接收行星常数和分子黏性系数，再读取一组 atke_* / iflag_* 配置键，生成后续 ATKE 交换系数、TKE 演化、地表层稳定度函数和 PBL 诊断共用的模块变量。

该文件本身不计算扩散通量；它的职责是把 ATKE 方案的开关、经验常数、稳定度函数参数和分子黏性系数集中初始化，并通过 SAVE + !$OMP THREADPRIVATE 的模块状态供下游读取。

共享状态与副作用

  • 模块状态：本文件所有 ATKE 参数均为 SAVE 模块变量，并声明为 !$OMP THREADPRIVATE。并行复现时要确认每个 OpenMP 线程都已经经过初始化或正确继承初始化值。
  • 配置读取：atke_ini 会读取 .def 键并覆盖默认值；这些值不会作为参数显式传给下游，而是由其他模块 use lmdz_atke_turbulence_ini, only: ... 读取。
  • 中止路径：当 iflag_atke==0 且 iflag_atke_lmix>0、cinf<=cn、pr_slope<=1、pr_asym>=pr_neut 时调用 abort_physic 中止。
  • 无文件 I/O：本文件不直接打开或写文件，也不输出诊断字段。

核心逻辑

  1. 复制调用方常数：atke_ini 先把 rg_in/rd_in/rpi_in/rcpd_in/rv_in/viscom_in/viscoh_in 写入同名模块缓存，供 ATKE 下游模块读取。
  2. 读取方案开关：依次读取 iflag_atke、iflag_atke_lmix、atke_ok_vdiff、atke_ok_virtual、iflag_num_atke。其中 iflag_atke==0 表示 stationary TKE 路径；该路径不允许使用依赖 TKE 的混合长度公式。
  3. 读取长度与临界参数：读取 atke_l0、atke_ric 和 atke_cepsilon，并由 cepsilon 计算 cn。
  4. 构造 ATKE 稳定度函数常数：读取 atke_cinffac，计算 cinf=cinffac*cn，要求 cinf>cn；再计算 ctkes=cepsilon**(2./3.)。
  5. 读取 Prandtl 数函数参数：读取 atke_pr_slope、atke_pr_neut、atke_pr_asym，要求 pr_slope>1 且 pr_asym<pr_neut。
  6. 读取混合长度和 TKE 扩散调参：读取 atke_clmix、atke_clmixshear、atke_smmin、atke_cke。
  7. 派生过渡尺度：计算 ri0=2./rpi*(cinf-cn)*ric/cn 和 ri1=-2./rpi*(pr_asym-pr_neut)，下游用它们在不稳定 Richardson 数区间平滑过渡。

  cn = (1 / sqrt(cepsilon)) ** (2/3)
  读取 cinffac

主题 参与方式
边界层与湍流 初始化 ATKE 1.5 阶湍流闭合方案所需的稳定度函数、TKE 耗散、混合长度和扩散参数

写法特点

  • 集中式初始化模块：本文件只初始化和保存参数，下游模块通过 use ... only 读取，不通过显式参数链传递。
  • SAVE + THREADPRIVATE：所有关键状态均为线程私有，适合 OpenMP，但要求初始化路径在并行使用前完成。
  • 配置键默认值写在源码中：例如 atke_l0=15.0、atke_ric=0.25、atke_cepsilon=5.87、atke_cinffac=2.0、atke_pr_neut=0.8。
  • 参数合法性直接中止：范围检查只覆盖影响公式一致性的组合；其他数值如 clmix、smmin、cke 没有额外范围保护。

复现要点

  • 只有 physiq_mod.F 中 callatke 为真时才调用 atke_ini；当前调用处把 viscom 和 viscoh 设为 0.001，并把 rv 传为 0.。
  • .def 中未提供键时使用源码默认值；复现实验必须记录 iflag_atke、iflag_atke_lmix、iflag_num_atke 和所有 atke_* 键。
  • iflag_atke==0 时不能设置 iflag_atke_lmix>0，否则初始化立即中止。
  • cinffac 必须让 cinf>cn；pr_slope 必须大于 1；pr_asym 必须小于 pr_neut。
  • atke_ok_vdiff 会打开 TKE 垂直扩散；atke_ok_virtual 会让下游 ATKE 计算考虑虚温/浮力修正。

待确认

  • abort_physic 在本文件中未显式 use，应由旧式隐式外部过程或构建环境提供；待确认具体接口来源。
  • 行 105 的 cn=(1./sqrt(cepsilon))**(2/3) 中 2/3 在 Fortran 中可能按整数除法求值为 0；若如此，cn 会变为 1，而注释意图更像 2./3.。这是复现风险，需要用实际编译器/告警选项确认。
  • lunout 和 prt_level 在本模块声明并设为 THREADPRIVATE，但本文件未赋值也未使用；待确认是否为历史遗留接口。
  • OpenMP 场景下 THREADPRIVATE 参数是否通过主线程初始化后复制到工作线程，取决于调用时序和编译运行设置。
