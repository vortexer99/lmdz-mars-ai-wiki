文件定位

tracer_mod.F90 是 LMDZ.MARS 的tracer 共享定义核心模块。它声明了所有 tracer 相关的全局状态变量（tracer 数量、名称、分子量、物理属性、tracer 索引），并提供初始化（ini_tracer_mod）和释放（end_tracer_mod）子程序。被 30+ 个物理模块使用，是整个 tracer 系统的枢纽。

共享状态与副作用

  • 所有变量都是 SAVE + !$OMP THREADPRIVATE：每个 OpenMP 线程有独立副本。
  • allocatable 数组：noms, mmol, radius, rho_q, alpha_lift, alpha_devil, igcm_dustbin, nqdust, nqchildren — 由 ini_tracer_mod 分配；end_tracer_mod 释放前 8 个，未释放 nqchildren（行 239-246 遗漏，疑似 bug）。
  • tracer 索引（igcm_*）：源码注释"should be 0 if the corresponding tracer does not exist"（行 42-43），但无显式 = 0 初始化；实际由 initracer.F90 根据 tracer 名称匹配赋值。
  • 物性常数（rho_dust, rho_ice 等）：由 initracer.F90 或 conf_phys.F 赋值。

核心逻辑

ini_tracer_mod（行 157–233）

  1. 查找配置文件（行 176–208）：
    • 先查 traceur.def（旧格式）→ 用 parse_traceur_file_mod 解析
    • 不存在则查 tracer.def（新格式）→ 用 parse_tracer_file_mod 解析
    • 都不存在 → abort_physic 终止
  2. 检查 CO2 tracer（行 210–221）：遍历所有 tracer 名称，必须有 "co2"，否则终止
  3. 分配数组（行 224–231）：分配 mmol, radius, rho_q, alpha_lift, alpha_devil, igcm_dustbin, nqdust

end_tracer_mod（行 235–248）

  • 释放 noms, mmol, radius, rho_q, alpha_lift, alpha_devil, igcm_dustbin, nqdust 这 8 个 allocatable 数组（带 allocated() 保护）；nqchildren 也是 allocatable，但源码未在此释放，已在待确认中标为疑似遗漏。

伪代码

ini_tracer_mod:
  if traceur.def exists:
    parse traceur.def → nqmx, noms, nqparents, nqchildren
  elif tracer.def exists:
    parse tracer.def → nqmx, noms, nqparents, nqchildren
  else:
    abort

  if "co2" not in noms:
    abort("co2 tracer is now mandatory")

  allocate mmol, radius, rho_q, alpha_lift, alpha_devil, igcm_dustbin, nqdust

end_tracer_mod:
  deallocate noms, mmol, radius, rho_q, alpha_lift, alpha_devil, igcm_dustbin, nqdust
  ! NOTE: nqchildren is NOT deallocated here — likely an oversight

写法特点

  • 纯声明模块：模块体只有变量声明，无可执行代码（除 contains 中的子程序）。
  • 所有变量 SAVE + !$OMP THREADPRIVATE：确保 OpenMP 并行安全。
  • 旧/新格式双轨：ini_tracer_mod 先查 traceur.def，不存在再查 tracer.def。
  • CO2 强制检查：co2 tracer 是必须的（行 219–221），反映 CO2 在火星大气中的核心地位。
  • tracer 索引由 initracer.F90 赋值：本模块只声明和分配，实际的索引匹配在 initracer 中完成。

复现要点

  • 必须有 traceur.def 或 tracer.def 文件，且包含 "co2" tracer。
  • tracer 物性（mmol, radius, rho_q 等）由 initracer.F90 赋值，不在本模块。
  • 所有 igcm_* 索引应为 0（未显式初始化），由 initracer.F90 根据 noms 匹配赋值。

待确认

  • igcm_ccnco2_meteor_mass/number（行 66–67）和 igcm_ccnco2_h2o_*（行 68–70）在 !$OMP THREADPRIVATE 声明中缺失（行 72–74 只列了部分），推断为遗漏（待确认：是否影响 OpenMP 并行）。
  • varian, r3n_q, rho_dust 等标量的赋值位置不在本文件（待确认：由 initracer 或 conf_phys 赋值）。
  • nqchildren（行 147）为 allocatable 数组，由 ini_tracer_mod 分配但 end_tracer_mod（行 239-246）未释放，疑似内存泄漏遗漏。
