phymars/dyn1d 子目录

路径

LMDZ.MARS\libf\phymars\dyn1d

定位

dyn1d 存放 1D testphys1d 运行路径需要的支撑文件：1D 主程序、初始化、垂直坐标、控制变量、串行/并行占位接口、profile 读取、restart 写出和少量公共模块。该目录服务于单柱物理测试，不是三维 GCM 主积分目录。

已建页面

文件 说明
abort_gcm.F 指向 LMDZ.COMMON/libf/dyn3d/abort_gcm.F 的中止例程路径 stub；供 dyn1d 调用 abort_gcm。
comconst_mod.F90 指向 LMDZ.COMMON/libf/dyn3d_common/comconst_mod.F90 的动力常数模块路径 stub；目标模块保存动力维度、时间步、行星常数、耗散和顶层 sponge 参数。
comvert_mod.F90 指向 LMDZ.COMMON/libf/dyn3d_common/comvert_mod.F90 的垂直坐标模块路径 stub；dyn1d 初始化和主循环用 ap/bp/aps/bps 构造 plev/play。
control_mod.F90 指向 LMDZ.COMMON/libf/dyn3d_common/control_mod.F90 的公共动力控制状态 stub；当前 dyn1d 目录无直接 use control_mod。
disvert_noterre.F 指向 LMDZ.COMMON/libf/dyn3d_common/disvert_noterre.F 的非地球垂直离散例程 stub；读取 esasig.def 或 z2sig.def 并写入 ap/bp/aps/bps/presnivs/pseudoalt。
ener_mod.F90 指向 LMDZ.COMMON/libf/dyn3d_common/ener_mod.F90 的公共动力能量诊断状态 stub；当前 dyn1d 目录无直接 use ener_mod。
infotrac.F90 指向 LMDZ.COMMON/libf/dyn3d_common/infotrac.F90 的 tracer 元数据模块 stub；dyn1d 初始化直接读 traceur.def 写入 nqtot/tname/nqperes/nqfils。
init_testphys1d_mod.F90 1D testphys1d 初始化主例程；读取配置、traceur.def、restart/profile，填充时间、垂直坐标、几何、坡面、地表、土壤、tracer 和水控制状态。
logic_mod.F90 指向 LMDZ.COMMON/libf/dyn3d/logic_mod.F90 的动力逻辑状态 stub；当前 dyn1d 路径直接使用 hybrid 控制垂直坐标。
mod_const_mpi.F90 指向 LMDZ.COMMON/libf/dyn3dpar/mod_const_mpi.F90 的并行常量 stub；提供 COMM_LMDZ/MPI_REAL_LMDZ 和 MPI/XIOS/OASIS 初始化入口。
mod_interface_dyn_phys.F90 指向 LMDZ.COMMON/libf/dynphy_lonlat/mod_interface_dyn_phys.F90 的动力-物理接口 stub；并行分支保存物理列到动力 (i,j) 的索引映射。
parallel_lmdz.F90 指向 LMDZ.COMMON/libf/dyn3dpar/parallel_lmdz.F90 的并行运行时 stub；初始化 MPI/OpenMP rank 分配、halo 交换和 gather/broadcast 接口。
profile_temp_mod.F90 1D 温度廓线构造模块；按 ichoice/tref/isin/pic/largeur/hauteur 生成初始温度，可读取 profile_temp 并写出 profile_temp.out。
read_profile_mod.F90 1D tracer 初始廓线读取模块；按 profile_<tracer> 文件填充 qsurf/q，并为 CO2、HDO 和 dust 缺文件路径提供默认值。
serre_mod.F90 指向 LMDZ.COMMON/libf/dyn3d_common/serre_mod.F90 的公共动力 zoom/水平网格状态 stub；当前 dyn1d 目录无直接 use serre_mod。
testphys1d.F90 1D 单柱物理测试主程序；检查运行文件、调用初始化、生成 startfi.nc、循环调用 physiq 并可写 restart1D.txt。
writerestart1D_mod.F90 1D 文本 restart 写出模块；按 ps、tracer、u、v、teta 顺序覆盖写出 restart1D.txt。
